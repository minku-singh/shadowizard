#What is this?

Get perfect shadows for elements

#Installation

`npm i shadowizard --save`

then...

```
import {shadowizard} from 'shadowizard';

shadowizard({
    shadow_type: 'soft',
    padding: false
});
```

## options

Shadowizard supports 2 options:
- *shadow_type* - _hard / soft_ (Default to soft)
- *padding* - _boolean_ (Default to false)
